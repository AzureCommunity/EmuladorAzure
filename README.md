# Requisitos
  
  -Visual Estudio 2015 o superior
  
  -Plugin GitHub para Visual Estudio (https://visualstudio.github.com/)
  
  -En cada Commit se tiene que poner los cambios realizados a grandes rasgos Ejemplo: "Traducción"
